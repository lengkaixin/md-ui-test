# lerna_demo

1. 统一发布所有子包
2. 单独发布子包
3. 发布整包

## Getting Started

Install dependencies,

```bash
$ npm i
```

```
// 为 packages 安装依赖

npm run bootstrap
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build` ,

```bash
$ npm run build
```

publish

```bash
$ npm run pub
```
